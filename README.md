# Online Examination System

## Description:
The Online Examination System is a web-based application designed to conduct examinations digitally. It allows users to take exams online, manages questions, evaluates answers automatically, and displays results efficiently. This system reduces manual effort and improves accuracy.

## Technologies Used:
HTML  
CSS  
JavaScript  


## Features:
User login and exam access  
Online question display  
Automatic evaluation of answers  
Instant result generation  
Simple and user-friendly interface  

## How to Run:
Open the index.html file in a web browser.

University mini project
